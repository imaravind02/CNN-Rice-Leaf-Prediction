# PRCP-1001 Rice Leaf Disease Detection

## Project Overview
This project focuses on **Rice Leaf Disease Detection** using Machine Learning and Deep Learning techniques. The goal is to classify rice leaf images into different disease categories using image processing and deep learning models.

### Diseases Covered
- Leaf smut  
- Brown spot  
- Bacterial leaf blight

## Problem Statement
